# 数学课程设计师的工具箱 (The Mathematics Curriculum Designer's Toolkit)

## 引言

本工具箱旨在为一线数学教育工作者和课程设计师提供一套实用、高效、富有启发性的资源。它不仅包含用于规划单元和课时的结构化模板，更通过详实的填写示例，展示了如何将先进的教育理念转化为具体的教学设计。此外，本工具箱还提供了能直接用于课堂、激活学生思维的轻量级工具。

---

## 第一部分：单元规划工具

### 工具一：UbD（理解为先）单元设计模板

**用途**：用于规划一个以培养学生深度理解和知识迁移能力为目标的教学单元。它倡导"逆向设计"，从最终目标（学生需要理解什么、能做什么）出发，倒推评估方式和教学活动。

| **阶段** | **核心要素** | **设计问题/内容** |
| :--- | :--- | :--- |
| **第一阶段：确定预期学习结果** | **核心理解 (Understandings)** | *学生将持久理解的"大概念"是什么？* |
| | **核心问题 (Essential Questions)** | *能引发探究、思考并贯穿整个单元的开放性问题是什么？* |
| | **学习目标 (Objectives)** | *学生将知道 (Know) ...*   *学生将能够 (Be able to do) ...* |
| **第二阶段：确定评估证据** | **表现性任务 (Performance Tasks)** | *学生需要完成什么复杂的、真实世界的任务来证明他们的理解？* |
| | **其他证据 (Other Evidence)** | *通过什么其他方式（如测验、观察、作业）来评估学习目标？* |
| **第三阶段：规划学习体验** | **学习活动 (Learning Plan - WHERETO)** | *WHERETO框架：* **W**-目标与原因 **H**-吸引学生 **E**-提供经验和工具 **R**-反思与修正 **E**-自我评估 **T**-个性化 **O**-组织有序 |

---

#### **填写示例：以初中"一次函数"单元为例**

| **阶段** | **核心要素** | **设计内容** |
| :--- | :--- | :--- |
| **第一阶段** | **核心理解** | 学生将理解： 1. 函数是描述现实世界中一个量如何依赖另一个量变化的核心数学模型。 2. 一次函数的图像（直线）及其斜率，直观地揭示了变化率的恒定性。 |
| | **核心问题** | 1. 我们如何用数学语言来预测未来？ 2. "变化"本身可以是恒定的吗？它在现实世界中是什么样的？ |
| | **学习目标** | **学生将知道**：一次函数的定义、图像、性质（k,b的意义）。 **学生将能够**：1.根据实际问题写出一次函数的解析式。2.在坐标系中画出一次函数的图像。3.利用一次函数模型解决简单的决策问题。 |
| **第二阶段** | **表现性任务** | **"选择最省钱的手机套餐"项目**： 学生分组研究本地两家不同运营商的手机套餐（包含不同的固定月租和通话/流量单价），建立函数模型，通过图像和计算分析在不同使用量下哪个套餐更划算，并最终以报告或演示的形式，向"家庭"（其他同学扮演）推荐最佳方案。 |
| | **其他证据** | 单元小测（考察基本概念和计算）、课堂练习（画图、求解）、教师对小组讨论的观察记录。 |
| **第三阶段** | **学习活动** | **W**: 单元开始时明确目标——我们将学习一种强大的工具来帮助我们做出明智的决策。 **H**: 提出问题："打车平台的两种计价方式，哪种对你更有利？"激发学生兴趣。 **E**: 教授函数定义、画图方法；提供坐标纸、在线画图工具（如Desmos）。 **R**: 学生在解决手机套餐问题时，需要不断反思、修正自己的模型。 **E**: 学生完成项目后，进行小组自评和互评。 **T**: 为不同水平的学生提供不同复杂度的套餐方案（如增加分段计价）。 **O**: 单元活动从具体问题到抽象模型，再回到具体应用，逻辑清晰。 |

## 第二部分：课时设计工具

### 工具二：5E探究式课时计划

**用途**：适用于设计一节以学生为中心、强调动手探究和意义建构的数学课。

* **吸引 (Engage)**: 如何激发学生的好奇心？（提出一个反常识的问题、展示一个有趣的现象、讲述一个故事）
* **探索 (Explore)**: 学生将进行什么动手操作或探究活动来收集数据、检验想法？（教师在此阶段应"管住嘴、迈开腿"，多观察，少讲解）
* **解释 (Explain)**: 学生如何分享他们的发现？教师如何在此基础上，引导学生形成共识，并引入正式的数学术语和概念？
* **迁移 (Elaborate)**: 学生如何将所学概念应用到新的情境中，以深化理解？
* **评估 (Evaluate)**: 如何检查学生是否达到了学习目标？（可以通过提问、出口票、作品等方式进行形成性评价）

---

#### **填写示例：以"圆的面积公式探索"课时为例**

* **吸引 (Engage)**:
  * **活动**: 教师展示两个大小不同的披萨盒，提问："如果我们想知道哪个披萨更'值'，只看直径够吗？我们需要比较它们的什么？我们该如何比较？"
* **探索 (Explore)**:
  * **活动**: 学生分组获得一个硬纸板做的圆形、剪刀、尺子、胶水。任务：**"请尝试将这个圆形剪拼成一个你熟悉的、会计算面积的图形（如长方形、平行四边形），并思考这个新图形的'底'和'高'与圆的什么有关？"** 学生将圆剪成8份、16份或更多份的扇形，并尝试将其拼接。
* **解释 (Explain)**:
  * **活动**: 各小组展示自己的拼接结果，会发现当扇形分得越细，拼成的图形越接近一个长方形。教师引导全班讨论并得出结论：
    * 这个"长方形"的**长**，约等于圆周长的一半 (`πr`)。
    * 这个"长方形"的**宽**，约等于圆的半径 (`r`)。
  * **关键概念**: 最终，师生共同推导出圆的面积公式 `S = πr * r = πr²`。
* **迁移 (Elaborate)**:
  * **活动**: 计算一个"花环"的面积（大圆面积减小圆面积），或解决一个"羊吃草"问题（绳子长度是半径，计算羊能吃到的草地面积）。
* **评估 (Evaluate)**:
  * **出口票 (Exit Ticket)**: "请用一两句话，并配上你自己的图，向一个没上过这节课的同学解释，为什么圆的面积公式是 `πr²`，而不是别的？"

## 第三部分：思维激活工具

### 工具三：优质提问设计清单

**用途**：在备课和教学中，用此清单检查自己的提问是否能激发学生的深度思考。

* [ ] **启动思考**: "关于这个问题，你首先想到了什么？" / "你注意到了什么？"
* [ ] **要求澄清**: "你能用自己的话再解释一遍吗？" / "能举个例子吗？"
* [ ] **探究原因**: "为什么会这样？" / "你这样想的理由是什么？"
* [ ] **寻找联系**: "这和你以前学过的什么知识有联系？" / "这和我们生活中的什么现象很像？"
* [ ] **考虑不同观点**: "有没有其他人有不同的想法？" / "你还能用别的方法解决这个问题吗？"
* [ ] **进行元认知**: "你是如何开始思考这个问题的？" / "在解决问题时，你在哪里遇到了困难？又是如何克服的？"
* [ ] **提出假设与预测**: "如果我们改变其中一个条件，结果会如何变化？" / "你对下一步的结果有什么猜想？"

### 工具四：数学讨论句式启动器

**用途**：打印出来贴在教室墙上，或放在学生桌上，帮助他们发起和参与有建设性的数学对话。

| **对话功能** | **句式启动器** |
| :--- | :--- |
| **表达同意** | "我同意 (同学名) 的观点，因为..."   "我的想法和 (同学名) 类似..." |
| **表达异议** | "我有一个不同的看法，我认为..."   "我理解你的思路，但是否忽略了...?" |
| **进行补充** | "我想在 (同学名) 的基础上补充一点..." |
| **提出疑问** | "我感到困惑的是..."   "你能再详细解释一下...的部分吗？" |
| **寻求澄清** | "你的意思是...吗？"   "你说的'它'，具体是指哪个部分？" |
| **建立联系** | "这让我想起了我们之前学过的..." |

## 第四部分：自我反思工具

### 工具五：课程设计自查清单

**用途**：在完成单元或课时设计后，用这些问题进行自我审视和改进。

* [ ] **目标的一致性**: 我设计的评估和教学活动，是否都明确地、强有力地指向我的核心学习目标？
* [ ] **认知的深度**: 我的设计是否超越了机械记忆和程序性计算，真正鼓励了学生的分析、评估、创造等高阶思维？
* [ ] **情境的真实性**: 我的课程内容是否与学生的现实生活或他们关心的真实世界问题产生了有意义的联系？
* [ ] **参与的公平性**: 我的教学活动是否足够主动、有趣，并能确保所有学生（无论水平如何）都能有意义地参与其中？
* [ ] **支持的差异化**: 我的设计中是否为不同起点的学生提供了灵活的学习路径、多样的资源和个性化的支持？
* [ ] **思想的衔接性**: 这个设计如何建立在学生已有知识之上，并为他们未来的学习埋下了哪些重要思想的种子？
* [ ] **实施的可行性**: 我设计的活动，在现有的时间、材料和技术条件下是真实可行的吗？
